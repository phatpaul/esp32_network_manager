Initialise the Wi\+Fi Manager.

Calling this function will initialise the Wi\+Fi Manger. It must be called after initialising the N\+VS, default event loop, and T\+CP adapter and before calling any other esp\+\_\+wmngr function. After successfull initialisation the Wi\+Fi Manager will be in the \char`\"{}stopped\char`\"{} state. It may then be configured with \mbox{\hyperlink{wifi__manager_8h_abbb9b00873b40b99e3d2b62b2ae1b058}{esp\+\_\+wmngr\+\_\+set\+\_\+cfg()}} or directly started using \mbox{\hyperlink{wifi__manager_8h_a60a0b9bc2c3230b518a242e8116c574c}{esp\+\_\+wmngr\+\_\+start()}}.

\end{DoxyReturn}

Set a new Wi\+Fi Manager configuration.

This function is used to set a new Wi\+Fi Manager configuration. The current configuration is backed up and an asynchronous update process is triggered.

If Wi\+FI Manager is in state \mbox{\hyperlink{wifi__manager_8h_acfa752b31a99436ae37776ac0124ccefac8a7d4bfd04b39821c4ec68f386a30b1}{wmngr\+\_\+state\+\_\+stopped}}, the new config will be applied once \mbox{\hyperlink{wifi__manager_8h_a60a0b9bc2c3230b518a242e8116c574c}{esp\+\_\+wmngr\+\_\+start()}} is called.

If setting the new configuration succeeds, the state reported by \mbox{\hyperlink{wifi__manager_8h_a0d05736f568d0e3671d5c6b5233dab5a}{esp\+\_\+wmngr\+\_\+get\+\_\+state}} will change to \mbox{\hyperlink{wifi__manager_8h_acfa752b31a99436ae37776ac0124ccefac9686d9646295e6c849054058601e8a3}{wmngr\+\_\+state\+\_\+connected}} (in S\+TA or A\+P\+S\+TA mode) or \mbox{\hyperlink{wifi__manager_8h_acfa752b31a99436ae37776ac0124ccefa41ca5963fd23eee073d864d3d4d551be}{wmngr\+\_\+state\+\_\+idle}} (in AP mode). If the new configuration fails, the device will revert to the previous configuration and set the state to \mbox{\hyperlink{wifi__manager_8h_acfa752b31a99436ae37776ac0124ccefa5de108d139875565ba4c04a6cb0f6331}{wmngr\+\_\+state\+\_\+failed}}.
